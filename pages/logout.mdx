# Logout

Logged in users can logout using the API described below. The logout operation requires two properties: one marked as user id identity and the second is token.

**Method :** <Badge>POST</Badge>

**URL :** <Badge variant="dark">/api/logout</Badge>

**Body :** <Badge variant="purple">JSON</Badge>

```json
{
	"user_id": 2,
	"token": "mdLDoIfP7zznp4cqQJ0ZHm-B1QxYA1y43DDqn2uYPmk"
}
```

**Request :**

<CodeGroup items={["Javascript", "Python"]}>
	<CodeGroup.Item>
	```jsx copy
	var myHeaders = new Headers();
    myHeaders.append("Content-Type", "application/json");

    var raw = JSON.stringify({
        "user_id": 2,
        "token": "mdLDoIfP7zznp4cqQJ0ZHm-B1QxYA1y43DDqn2uYPmk"
    });

    var requestOptions = {
        method: 'POST',
        headers: myHeaders,
        body: raw,
        redirect: 'follow'
    };

    fetch("/api/logout", requestOptions)
        .then(response => response.text())
        .then(result => console.log(result))
        .catch(error => console.log('error', error));
	```
	</CodeGroup.Item>
	<CodeGroup.Item>
	```python copy
	import http.client
import json

conn = http.client.HTTPSConnection("localhost", undefined)
payload = json.dumps({
  "user_id": 2,
  "token": "mdLDoIfP7zznp4cqQJ0ZHm-B1QxYA1y43DDqn2uYPmk"
})
headers = {
  'Content-Type': 'application/json'
}
conn.request("POST", "/api/logout", payload, headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))
	```
	</CodeGroup.Item>
</CodeGroup>
